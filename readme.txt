Copy the "seplugins" folder and the "rg" folder in the root of your memory stick.
Then, enable the plugin and enjoy!

The root of the gameboots are : 

1 -> ms0:/psp/system/rg/gameboot.pmf
2 -> flash0:/vsh/resource/gameboot.pmf
3 -> flash1:/vsh/resource/gameboot.pmf

For example : 

Us the 3.XX Recovery menu and toogle usb flash1 to put the gameboot on flash1:/vsh/resource (make the folder).
